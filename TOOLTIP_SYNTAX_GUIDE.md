# Tooltip Syntax Guide - Easy Inline Tooltips

## üöÄ **Super Easy Syntax: `[[[term|definition]]]`**

Instead of writing complex HTML, just use this simple syntax:

```
[[[term|definition]]]
```

## üìù **Examples**

### **Basic Usage:**
```
Welcome to [[[Darjeeling|A hill station in India famous for tea production]]] tea.
```

**Result:** "Welcome to Darjeeling tea." (where "Darjeeling" has a tooltip)

### **Multiple Tooltips:**
```
[[[Black tea|Fully oxidized tea with dark color]]] and [[[Green tea|Minimally processed tea that retains green color]]] are both popular.
```

**Result:** Both "Black tea" and "Green tea" have tooltips

### **With HTML in Definitions:**
```
[[[Matcha|Japanese powdered <strong>green tea</strong> used in ceremonies]]]
```

**Result:** Tooltip can include HTML formatting

## üîß **Syntax Rules**

- **Format:** `[[[term|definition]]]`
- **Separator:** Use `|` (pipe) between term and definition
- **Brackets:** Exactly 3 opening `[[[` and 3 closing `]]]`
- **Spaces:** OK to have spaces around the pipe: `[[[ term | definition ]]]`
- **HTML:** Can include HTML in definitions: `<strong>`, `<em>`, `<br>`

3. **Tooltip text:** Appears in golden yellow (`#ffbb00`)

## üí° **Pro Tips**

1. **Keep definitions concise** - 1-2 sentences work best
2. **Use clear language** - Avoid jargon in definitions
3. **Test on mobile** - Tooltips work by tapping on phones
4. **Don't overuse** - Too many tooltips can be distracting
5. **Be consistent** - Use the same definition for the same term
